# 公共看板与线路检测

## 公共看板

匿名访问 `/` 时会直接进入公共看板，不需要登录。只有管理员显式开启公开展示的节点会出现在这里。

公共数据只包含展示名称、在线状态、数据新鲜度、CPU、内存、磁盘和允许公开的线路检测图表。主机地址、端口、令牌、用户关系和管理操作不会对匿名访问者暴露。

## 设置公开节点

进入设置里的“公共看板”，选择需要展示的节点。可以设置公开名称，并选择是否展示 CPU、内存和磁盘摘要。

## 线路检测

进入设置里的“线路检测”配置线路。每条线路包含名称、检测类型、目标、间隔、超时、启用状态和公开展示状态。

支持的检测类型：

- TCPing：目标格式为 `host:port`。
- ICMP Ping：目标格式为不带端口的主机名或 IP。
- HTTP GET：目标格式为 `http://` 或 `https://` URL。

默认情况下，线路会由所有可用执行节点定期检测。需要精确指定执行节点时，在“高级设置”中选择。离线或不支持检测的执行节点会记录失败结果，不会阻塞其他线路。

## 公开线路图表

当执行节点本身公开展示时，线路图表默认会显示在公共看板中。如果某个目标或走势不适合公开，可以关闭该线路的公开展示。登录用户仍可在后台按权限查看检测配置和结果。
